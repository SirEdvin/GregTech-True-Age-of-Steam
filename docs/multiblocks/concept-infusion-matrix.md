Concept infusion matrix is first HV multiblock that is designed as little logistic puzzle game that rewards player with concept catalyst - item, that can be used to infuse conceptualized steel or air with said concept. 

This multiblock is a second major unlocker for all other multiblocks past [Infernal boiler](./infernal-boiler.md).

## How it works

While building concept infusion matrix, you need to place 8 same single-block MV machines inside. To perform crafting, this machines need to run all time, from crafting start and until crafting end, otherwise the blank catalyst will be voided and you need to start crafting all over again.


??? tip
    Concept infusion matrix has 6 places to put diode/passthrought hatches. Use them wisely.
